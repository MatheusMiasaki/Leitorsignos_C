# Leitor de Signos em C
## Matheus Miasaki E Rodrigo Furlaneto

  Este projeto consiste em uma aplicação simples que recebe a data de nascimento do usuário e identifica automaticamente seu signo do zodíaco. Após o processamento, o sistema abre uma página em HTML personalizada, exibindo o signo correspondente, juntamente com uma imagem ilustrativa e uma breve descrição de suas principais características.
